## 🛠️ Tecnologias e Pilares Aplicados

Para a construção deste portfólio acadêmico, utilizei os 4 pilares do Pensamento Computacional:

1.  **Decomposição:** Fragmentação de sistemas complexos em microserviços.
2.  **Reconhecimento de Padrões:** Uso de normas de segurança e design de software.
3.  **Abstração:** Foco nos requisitos essenciais para escalabilidade.
4.  **Algoritmos:** Lógica estruturada para automação de processos.
